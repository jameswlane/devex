# Web Development Environment

This example provides a complete modern frontend development environment suitable for React, Vue, Angular, and other web frameworks. It includes essential tools, editors, browsers, and development utilities.

## Quick Start

<Steps>
  <Step>
    ### Download Configuration
    
    ```bash
    # Download the web development configuration
    curl -fsSL https://raw.githubusercontent.com/jameswlane/devex/main/examples/web-development.yaml > ~/.devex/web-dev.yaml
    
    # Preview what will be installed
    devex config apply --file ~/.devex/web-dev.yaml --dry-run
    ```
  </Step>
  
  <Step>
    ### Apply Configuration
    
    ```bash
    # Apply the configuration
    devex config apply --file ~/.devex/web-dev.yaml
    
    # Install all tools
    devex install
    ```
  </Step>
  
  <Step>
    ### Verify Installation
    
    ```bash
    # Check installed tools
    node --version
    npm --version
    code --version
    git --version
    ```
  </Step>
</Steps>

## Development Workflows

### Project Setup Automation

Create a script to automate new project setup:

```bash
#!/bin/bash
# ~/.devex/scripts/new-web-project.sh

project_name="$1"
framework="${2:-react}"

if [ -z "$project_name" ]; then
    echo "Usage: new-web-project <project-name> [react|vue|angular]"
    exit 1
fi

echo "🚀 Creating new $framework project: $project_name"

case $framework in
    "react")
        npx create-react-app "$project_name" --template typescript
        cd "$project_name"
        npm install -D tailwindcss postcss autoprefixer
        npx tailwindcss init -p
        ;;
    "vue")
        npm create vue@latest "$project_name" -- --typescript --router --pinia --vitest --cypress
        cd "$project_name"
        npm install
        ;;
    "angular")
        ng new "$project_name" --routing --style=scss
        cd "$project_name"
        ng add @angular/material
        ;;
esac

# Initialize git repository
git init
git add .
git commit -m "Initial commit"

# Open in VS Code
code .

echo "✅ Project $project_name created successfully!"
```

### Development Server Management

```bash
#!/bin/bash
# ~/.devex/scripts/dev-servers.sh

case "$1" in
    "start")
        echo "🚀 Starting development servers..."
        
        # Start databases if needed
        devex service start redis postgresql
        
        # Start main application
        if [ -f "package.json" ]; then
            npm run dev &
        fi
        
        # Start additional services
        if [ -f "docker-compose.dev.yml" ]; then
            docker-compose -f docker-compose.dev.yml up -d
        fi
        ;;
        
    "stop")
        echo "🛑 Stopping development servers..."
        pkill -f "npm run dev"
        docker-compose -f docker-compose.dev.yml down 2>/dev/null || true
        devex service stop redis postgresql
        ;;
        
    "restart")
        $0 stop
        sleep 2
        $0 start
        ;;
esac
```

## Troubleshooting

### Common Issues

**Node.js Version Conflicts:**
```bash
# Check current Node version
node --version

# List installed versions
mise list node

# Switch to LTS version
mise use node@lts
mise global node@lts
```

**Package Installation Failures:**
```bash
# Clear npm cache
npm cache clean --force

# Clear pnpm cache
pnpm store prune

# Reinstall node_modules
rm -rf node_modules package-lock.json
npm install
```

**VS Code Extension Issues:**
```bash
# Reset VS Code extensions
code --disable-extensions
code --list-extensions --show-versions

# Reinstall essential extensions
devex vscode setup-extensions
```
