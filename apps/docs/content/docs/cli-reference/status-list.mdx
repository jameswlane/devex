# devex status & list

The `status` and `list` commands provide essential information about your development environment. Use `status` to check what's installed and working, and `list` to browse available applications and discover new tools.

## devex status

Check the installation status of applications in your development environment.

### Usage

```bash
devex status [flags]
```

### Options

| Flag | Short | Type | Default | Description |
|------|-------|------|---------|-------------|
| `--format` | `-f` | `string` | `table` | Output format (table, json, yaml, csv) |
| `--category` | `-c` | `string` | `all` | Filter by category |
| `--verbose` | `-v` | `bool` | `false` | Show detailed information |
| `--check-updates` | `-u` | `bool` | `false` | Check for available updates |
| `--problems-only` | `-p` | `bool` | `false` | Show only problematic installations |

## devex list

Browse and discover available applications that can be installed with DevEx.

### Usage

```bash
devex list [flags]
```

### Options

| Flag | Short | Type | Default | Description |
|------|-------|------|---------|-------------|
| `--category` | `-c` | `string` | `all` | Filter by category |
| `--search` | `-s` | `string` | | Search by name or description |
| `--installed` | `-i` | `bool` | `false` | Show only installed applications |
| `--available` | `-a` | `bool` | `false` | Show only available (not installed) applications |
| `--format` | `-f` | `string` | `table` | Output format (table, json, yaml, list) |
| `--platform` | `-p` | `string` | `auto` | Filter by platform (linux, macos, windows) |
| `--detailed` | `-d` | `bool` | `false` | Show detailed information |

## Integration Examples

### Scripting with Status Commands

```bash
#!/bin/bash
# Check if required tools are installed

required_apps=("git" "docker" "node")
missing_apps=()

for app in "${required_apps[@]}"; do
    if ! devex status --format json | jq -r ".applications[] | select(.name == \"$app\" and .status == \"installed\") | .name" | grep -q "$app"; then
        missing_apps+=("$app")
    fi
done

if [ ${#missing_apps[@]} -ne 0 ]; then
    echo "Missing required applications: ${missing_apps[*]}"
    echo "Installing missing applications..."
    devex add "${missing_apps[@]}" --install
fi
```

### Monitoring and Alerting

```bash
# Check for problems and send alerts
problems=$(devex status --problems-only --format json)
if [ "$problems" != "[]" ]; then
    echo "$problems" | mail -s "DevEx Issues Detected" admin@company.com
fi

# Check for updates weekly
if [ "$(date +%u)" -eq 1 ]; then  # Monday
    updates=$(devex status --check-updates --format json)
    if [ "$updates" != "[]" ]; then
        echo "Updates available: $updates"
    fi
fi
```
